
Word Fly - Static HTML Version
==============================

This folder contains a simple HTML version of your Word Fly landing page that you can upload directly to any web server.

Files included:
- index.html (main page)
- styles.css (all styling)
- favicon.png (website icon - you need to add this)
- word-fly-character.png (main character image - you need to add this)

To set up:
1. Copy your Word Fly character image and name it "word-fly-character.png"
2. Copy your favicon image and name it "favicon.png"
3. Upload all files to your web server
4. Update the download button links in index.html to point to your actual app store pages

Features:
- Fully responsive design
- Works on all modern browsers
- No JavaScript dependencies
- Fast loading
- SEO optimized

To customize:
- Edit the download button links (href="#") to your actual app store URLs
- Modify colors in styles.css if needed
- Update text content in index.html as needed
